# comparing_field_with_iNat_interactions

This repo compares field collected plant-pollinator interactions with iNaturalist plan-pollinator interactions at 8 different parks in Broward County, Florida, USA. We have data from Marquis et al. (2026) for our field data and from Lilkendey et al. (2026) for our iNaturalist data.
